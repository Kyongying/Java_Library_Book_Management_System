# Java Library Book Management System

A Java-based Library Book Management System developed for a Data Structures and Algorithms course assessment. This project demonstrates the implementation of data structures, searching algorithms, and sorting algorithms in a practical library management application.

## Features

* Add new books
* View all books
* Search books by title, author, or ISBN
* Sort books by title, author, or book ID
* Issue and return books
* Update book information
* Delete book records

## Data Structure

* List (Java Collections Framework)

## Searching

* Search by title
* Search by author
* Search by ISBN

## Sorting

* Sort by title
* Sort by author
* Sort by book ID

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Data Structures (List)
* Searching Algorithms
* Sorting Algorithms

## Learning Outcomes

* Apply Java programming concepts
* Implement data structures
* Use searching and sorting algorithms
* Develop a functional information system
